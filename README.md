# 🚀 Customer Segmentation using Machine Learning

Customer Segmentation is a machine learning project designed to group customers into different clusters based on their purchasing behavior, demographics, and spending patterns. The project uses unsupervised learning techniques to identify hidden patterns in customer data and generate meaningful business insights.

---

# 🎯 Project Objectives

- Analyze customer behavior and spending patterns
- Perform data preprocessing and data analysis
- Apply clustering algorithms for customer segmentation
- Visualize customer groups and cluster characteristics
- Help businesses create targeted marketing strategies

---

# ✨ Features

- Data cleaning and preprocessing
- Data analysis using charts and plots
- Customer segmentation using K-Means Clustering
- Elbow Method to determine the optimal number of clusters
- Cluster visualization using 2D plots
- Business interpretation of each customer segment
